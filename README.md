# Ableton About Page Clone

A responsive, pixel-perfect clone of the [Ableton About page](https://www.ableton.com/en/about/), built as a challenge from [Frontend Practice](https://www.frontendpractice.com/).

This project focuses on translating a complex, design-heavy layout into clean code, emphasizing precise typography, asymmetrical image galleries, and interactive navigational elements.

**Live Demo:** [View the Ableton Clone](https://sahlminkok.github.io/ableton/)

## 🚀 Tech Stack
* **HTML5:** Semantic structuring.
* **CSS3:** Flexbox, CSS Grid, and custom `linear-gradient` backgrounds for complex section layouts.

## ✨ Key Features
* **Sticky Navigation:** A secondary header that locks to the top of the viewport on scroll using `position: sticky`.
* **Asymmetrical Layouts:** Recreated overlapping image galleries and distinct background color blocks.
* **Modern Typography:** Implemented the 'Jost' typeface to match Ableton's clean brand aesthetic.
